## Agent skills

### Issue tracker

Issues and PRDs are tracked in Linear using the generic Linear workflow; external PRs are not a triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Triage roles use default label strings: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

This repo uses single-context domain docs: root `CONTEXT.md` plus root `docs/adr/`. See `docs/agents/domain.md`.
